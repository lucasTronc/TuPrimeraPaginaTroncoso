# TuPrimeraPaginaTroncoso
La vista index muestra links a cada vista para que sea mas facil la navegacion.
Se armaron tres clases, usuario, categoria y post.
Se puede ingresar datos a la base a traves de formularios, los usuarios se ingresan en un vista y 
se muestran todos en otra, hay una vista para crear un post y otra para buscarlo en la base, y se puede agergar datos de categorias.
